# Contributing

Thank you for your interest in OpenVisionX.

## Development Workflow

1. Fork the repository.
2. Create a feature branch.
3. Write tests.
4. Pass Ruff.
5. Pass Pytest.
6. Submit a Pull Request.

## Coding Style

- Python 3.12
- Type Hints
- Google Docstring
- Ruff
- Pytest
